The main point:
The point of this script is to give the user a good time and date picker that is easy to navigate in a dialog.
From the programming standpoint it will give you a normal date that humans like and a mysql date.
I recommend having your pretty date text box visible and your mysql date hidden.

<input type="text" value="" id="prettydate" /> 

<input type="hidden" value="" id="mysqldate" /> 



You will need to have the standard jquery includes and the slick dtp includes:


<script src="http://code.jquery.com/jquery-1.10.2.js"></script>
<script src="http://code.jquery.com/ui/1.11.4/jquery-ui.js"></script>
<link rel="stylesheet" href="http://code.jquery.com/ui/1.11.4/themes/smoothness/jquery-ui.css">
<link rel="stylesheet" href="slick_dtp.css">
<script src="slick_dtp.js"></script>



You will need to assign it to your input ids on ready:


<script>
$(function() {
 
var slickDTP = new SlickDTP();
 
$( "#mysqldate" ).click(function() {
slickDTP.pickDate('#mysqldate','#prettydate');
});
 
$( "#mysqldate2" ).click(function() {
slickDTP.pickDate('#mysqldate2','');
});
});//end ready
</script>
